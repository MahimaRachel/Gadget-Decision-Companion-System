Data Flow Diagram (DFD)

Level 0 (Context Level)


                    User
                    │
                    │ Select domain
                    │ Select models
                    │ Set weights
                    ▼
                    Digital Gadget Decision Companion
                    │
                    │ Fetch ratings
                    │ Compute scores
                    ▼
                    User (Results + Recommendation)

Level 1

                    User
                    │
                    ▼
                    [Select Domain]
                    │
                    ▼
                    Fetch parameters from "domains" table
                    │
                    ▼
                    [Select Models]
                    │
                    ▼
                    Fetch ratings from "ratings" table
                    │
                    ▼
                    [Set Priority Weights]
                    │
                    ▼
                    calculate_decision()
                    │
                    ├─ Normalize weights
                    ├─ Apply custom ratings (if any)
                    ├─ Apply DB ratings
                    ├─ Apply fallback (5/10)
                    │
                    ▼
                    Rank models
                    │
                    ▼
                    Display:
                    - Top Recommendation
                    - Full Ranking
                    - Issues & Fixes
